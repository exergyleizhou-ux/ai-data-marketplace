## 5. Gotchas
- 种子数据集有**循环外键**(datasets↔dataset_versions):insert datasets 时
  current_version_id 留 NULL → 插 version → 插 file → 再 `UPDATE current_version_id`。
- offer `review_output:false` → job 自动放行(无需 ops release);job 证书响应字段
  是 `data.certificate_id`。
- compute 证书按**重算下载输出 SHA-256 == 证书 output_sha256** 验证
  (`/verify/:cert` 也支持)。
- **并发会话**会动记忆和 repo(Lumen 那条线是另一个会话)——push 前
  `git pull --rebase`,记忆分段写别整行覆盖。
- bash 里 `cd` 复合命令可能触发权限弹窗;尽量用绝对路径 / `go -C` / 子 shell。
- 后端 worktree `oasis-run` 标了 `prunable`(别 prune,它在跑 live)。
