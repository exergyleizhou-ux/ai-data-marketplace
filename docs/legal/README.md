# 法律文本(草案)与产品落点

> ## ⚠️ 重要声明
> 本目录下的三份文本均为 **草案骨架(v0.1)**,目的是**为执业律师节省起草时间**。
> - **这不是法律意见,未经审核不得使用、不得对外展示、不具法律效力。**
> - 必须由懂《个人信息保护法》《数据安全法》《网络安全法》《电子商务法》《民法典》合同编的**执业律师**起草/审核定稿。
> - 文中 `[方括号]` 为需填写/确认的占位符;⚠️ 标注为需律师重点裁定的条款。

## 三份文本

| 文件 | 作用 | 主要法律依据 |
|------|------|------------|
| [用户服务协议-草案](用户服务协议-草案.md) | 平台与用户(买/卖家)之间的总服务条款 | 民法典合同编、电子商务法 |
| [隐私政策-草案](隐私政策-草案.md) | 个人信息收集/使用/共享/存储/权利 | 个人信息保护法(PIPL)、数据安全法 |
