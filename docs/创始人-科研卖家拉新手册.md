## 1. 目标卖家画像(ICP)

一个"好卖家"=**手里有别人想要、但市面上拿不到「可信版本」的科研数据**。优先级从高到低:

| 级别 | 谁 | 为什么是好目标 | 在哪找 |
|---|---|---|---|
| **A 温线索** | PaperGuard 的用户 / star / issue 提过的人 | 已经信任你做"科研数据完整性";最低摩擦 | PaperGuard 的 GitHub stargazers / issues / PyPI 下载来源 / JOSS 评审与引用 |
| **A 温线索** | 你认识的实验室数据负责人 / 师兄弟 / 合作者 | 有信任、能直说 | 你的科研圈、导师网络 |
| **B 冷但精准** | 在 Zenodo / Figshare / Dryad / OSF 发过数据集的作者 | 已经愿意公开数据 = 愿意分享;只是没"可信版本" | 按领域搜这些库的高下载数据集 → 找通讯作者 |
| **B 冷但精准** | 在 Kaggle / UCI / HuggingFace 挂过数据集的人 | 同上,且习惯"数据当产品" | 数据集页的作者主页 |

## 4. 异议处理(诚实口径)

| 卖家说 | 你怎么答(别 over-claim) |
|---|---|
| "我的数据没问题,为什么要验证?" | 验证不是质疑你——是给**买家**一个不必信任你也能自查的凭证。你知道数据好,但你没法**证明**给一个陌生买家看。证书就是那个证明。 |
| "上传安全吗?会泄露吗?" | 体检在网络隔离沙箱里跑(`--network=none`),**原始数据行不出去,只走出聚合报告**;敏感数据还能用"可用不可见"——算法进沙箱、只取结果。 |
| "证书到底证明了什么?" | 证明**溯源 + 完整性**:这份内容(SHA-256 指纹)在这个时间、用这个钉死版本的算法、跑出了这个报告,任何人可重算比对。**它不证明数据"绝对正确"或"版权 100% 干净"**——我们有专门的诚实页说清边界。 |
| "标'异常'是不是说我数据是假的?" | 不是。标异常 = "看这里",不是"造假"。末位数堆叠、四舍五入、固定精度在真实测量里很正常;每条发现都附≥3 个无害解释。诚实是信任市场的命。 |
| "和 Kaggle / HuggingFace 免费数据有啥区别?" | 那些是**免费堆数据**,没人为可信背书;我们是**可信货架**——每份都体检 + 带可重算证书。买家为"放心"付费,这正是差异点。 |
| "谁会买?有买家吗?" | 诚实说:**冷启动阶段先靠卖家侧的验证价值**(你立刻拿到证书 + 招牌),撮合是第二幕。所以第一步**不绑定你卖出去**——先拿证书,挂着等。 |
| "我能赚多少?" | 见 §5。验证/上架几乎零成本;成交后平台抽 10%,其余归你。定价你自己定。 |
| "要花我多少时间?" | 上传一份 CSV + 填几行说明,体检自动跑,几分钟。我可以全程陪你跑第一份。 |

## 6. 演示脚本(3 分钟,让对方"看见")

> 现在就能演示。本地起好后给对方看;VPS 上线后换成公网稳定链接(见 §8)。

1. **打开市场列表**(`/datasets`)。"这是一个**只放可信数据**的市场。现在挂着 8 个真实科研
   数据集,横跨植物学、食品化学、材料、肿瘤医学、海洋生物、农学、环境、细胞生物——
   每个都跑过平台自己的完整性体检。"
2. **指徽章的颜色**。"注意徽章不是清一色绿。鸢尾花(Iris)、大肠杆菌(Ecoli)是**纯净绿洲**(clean);
   小麦籽粒、乳腺癌诊断是**建议复核**;红酒、玻璃、鲍鱼、森林火灾被标**显著异常**。
   我们**不盖橡皮图章**——如实报告。这恰恰是可信的来源。"
3. **点开一个 suspect 的**(如森林火灾)。"看,它具体标了哪一列、用了什么方法(Benford / 末位数 /
   哨兵值)、还附了无害解释。标异常 = 看这里,不是判造假。"
4. **打开证书**。"每份带一张证书,绑定内容指纹(SHA-256)。买家下载后**重算 SHA-256 一比就知道
   没被换过**。比如鸢尾花的证书号是 `VO-4911E9C918AF`——这串是用 数据集ID + 内容指纹 算出来的,
   谁都能重算。" (本地可现场演示重算,见下)
5. **收尾**:"把你的数据放进来,就长这样:自动体检 + 一张谁都能验的证书 + '已验证'徽章。
   我们现在就挂一份你的?"

**现场重算证书(可选,硬核买家会服):**
```bash
# 取这份数据的内容指纹(证书里的 content_sha256)与数据集 ID,然后:
printf '%s' "<datasetID>:<content_sha256>" | shasum -a 256 | awk '{print "VO-"toupper(substr($1,1,12))}'
# 输出应等于证书上的编号。
```

## 7. 上架转化清单(从"愿意"到"已上架✅")

- [ ] 拿到对方一份数据(CSV / 表格)+ 一句数据说明 + 来源/许可。

## 9. 诚实边界(别越界,信任市场全靠这个)

- 证书证 **溯源 + 完整性**(可重算指纹 + 钉死算法 + 时间),**不证**数据"正确"或"版权干净"。
- **公开可验证**需要 VPS 上线(临时隧道在本网络不稳)。上线前演示用本地 / 录屏;别承诺"现在就能公网验"。
- "标异常 ≠ 造假";报告如实记录操作特性,不下结论。
- 抽成 10%、Verify 定价如 §5——别临场加码或含糊。
- demo 的 8 个数据集是 **真实公开 UCI 数据**(CC BY 4.0,带原始引用);别把它们说成"独家"或"客户数据"。
